# Challenge-06

## packages
- express
- nodemon
- pg
- sequelize
- JWT
- Bcrypt
- BcryptJS
- swagger ui express

## Step to run
- clone the repositories
  ```
  git clone https://github.com/anugrahkresnaya/challenge-06.git
  ```
- install the dependencies
  ```
  npm install or yarn install
  ```
- set config for database
  ```
  setting config.json
  sequelize db:create
  sequelize db:migrate
  sequelize db:seed:all
  ```
- run the server
  ```
  npm run dev
  ```
